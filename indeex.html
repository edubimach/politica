Política de Privacidade – LinkedIn Auto Connect Última atualização: [insira a data de hoje] A extensão LinkedIn Auto Connect foi desenvolvida com o objetivo exclusivo de automatizar o envio de solicitações de conexão no LinkedIn, permitindo ao usuário incluir uma mensagem personalizada. Esta política explica como tratamos os dados dos usuários. 📌 Coleta de Dados Esta extensão não coleta, armazena, processa ou transmite quaisquer dados pessoais ou de navegação dos usuários. ⚙️ Funcionamento da Extensão A extensão atua localmente no navegador do usuário, com as seguintes funções: Identifica botões de conexão na interface do LinkedIn. Executa cliques programados para envio de solicitações de conexão. Permite adicionar mensagens personalizadas via popup da extensão. Armazena a mensagem temporariamente usando chrome.storage apenas durante a execução. Nenhuma informação é enviada ou recebida por servidores externos. 🔐 Segurança A extensão utiliza apenas permissões necessárias (activeTab, tabs, scripting, storage, host_permissions para LinkedIn) e não acessa nenhum outro site ou sistema. 👤 Dados de Identificação Nenhum dado pessoal do usuário, perfil do LinkedIn, ou informações de login são acessados, lidos ou manipulados pela extensão. 🧾 Alterações Esta política pode ser atualizada em caso de mudanças no funcionamento da extensão.
